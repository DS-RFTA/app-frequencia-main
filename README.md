# 📊 Sistema de Gestão - Ponto de Pregação Infantil

Um aplicativo web interativo desenvolvido em Python para digitalizar, armazenar e analisar os relatórios semanais dos pontos de pregação infantil.

## 🎯 Objetivo do Projeto
Substituir o preenchimento manual de relatórios em papel por um sistema digital robusto, permitindo o armazenamento seguro em banco de dados relacional e a visualização estratégica de indicadores (KPIs) por meio de um Dashboard dinâmico.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python
* **Interface Web:** Streamlit
* **Banco de Dados:** SQLite (nativo do Python)
* **Manipulação de Dados:** Pandas
* **Visualização de Dados:** Plotly

## 🚀 Funcionalidades
1. **Formulário de Inserção:** Coleta de dados sobre público local, componentes, vidas evangelizadas e materiais entregues.
2. **Armazenamento Seguro:** Inserção automática de dados estruturados utilizando a linguagem SQL.
3. **Dashboard Estratégico:** * Acompanhamento de métricas-chave (Total de Vidas, Conversões, Materiais).
   * Gráficos interativos para análise de perfil e evolução temporal.
   * Tabela de dados brutos para auditoria ou exportação.
